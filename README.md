# RMI Example in Java

ArithmeticImpl stellt über die RMIRegistry eine Instanz als ArithmeticServer zur Verfügung.
Client holt sich diese Instanz und spricht sie über das vereinbarte Interface IArithmetic an.

## Build

```batch
compile
```

## Run

```batch
startRMIRegistry
```

```batch
startServer
```

```batch
startClient
```
